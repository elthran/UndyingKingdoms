New Updates for Version 0.0.04

UI Improvements


  • Buying buildings and troops now properly updates total cost before you buy
  • New login page
  • Website now supports mobile on most pages
  • News feed can now show old news

New Class Features

  • Each county leader can choose a class: the leader's class affects what their county excels at
  • Starting with four classes:
  • - Warlord
  • - Rogue
  • - Engineer
  • - Merchant

Balance Changes

  • Attacking with a much weaker/strong army no longer decimates your army, but instead has them flee
  • Thief durations for missions has been adjusted. Getting caught has a much larger penalty.
  • Building costs updated. Thieves guilds are now more costly; houses and food production are cheaper

Bug Fixes

  • Thieves are now working as intended and return as intended
  • Leaderboard now properly adds score from traveling armies


Previous Updates from Version 0.0.03


New Infiltration mechanic

  • You now get 1 thief per thieves guild
  • You can send thieves on various missions: stealing gold, burning crops, spying
  • Thieves not on active missions will protect against enemy thieves

Inbox System

  • You will now be able to send messages to other players directly

New Racial Features

  • Each race will have several strengths/weaknesses, as well as different troop types
  • The county creation page will give some detail about races

Balance Changes

  • Land gained from attacking is now proportional to your land. So attacking counties weaker than you gains very little land.
  • Hunger has become more meaningful.
  • Excess production now gives much less gold

Quality of Life Changes

  • Added happiness and hunger breakdowns to the economy page so you can better manage your county
  • Attack page now lets you select troops in 10% increments, rather than listing every possible number of troops
  • Chatroom (Townhall) now auto-refreshes every few seconds and is more user-friendly

Bug Fixes

  • You can no longer attack yourself (not sure why anyone was trying to do that!)
